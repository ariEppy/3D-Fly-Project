To run the project- in command prompt-

Backend-
1. Build the Docker image-
docker build -t demo-app .

2. Run the container-
docker run -p 8080:8080 demo-app

Frontend-
3. Navigate to the frontend files-
cd frontend

4. navigate again into the frontend folder-
cd frontend

5. Run - 
npm install

6. Run - 
npm run dev
